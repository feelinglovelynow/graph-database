## üôè Description
* An open source graph database built with [Cloudflare Durable Objects](https://developers.cloudflare.com/durable-objects/) and [Cloudflare Workers](https://developers.cloudflare.com/workers/)
* Dream is for ***Graph Database*** to be a product that creates & unites lovely features inspired by [Prisma Studio](https://www.prisma.io/docs/orm/tools/prisma-studio), [Dgraph](https://dgraph.io/docs/dql/dql-get-started/) and [Cloudflare Durable Objects](https://developers.cloudflare.com/durable-objects/)
* So a [$5 monthly bill to Cloudflare](https://developers.cloudflare.com/durable-objects/platform/pricing/) with our open source ***Graph Database***  provides:
   * 1 million requests a month
   * Transactions
   * Indexes
   * Schema
   * Type safe queries and mutations
   * Generated jsdoc type definitions
   * UI to see / edit data
   * Data organized as a graph => no tables => no table joins => nested reads are fast
